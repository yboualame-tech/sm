# Student Management Application - Enhanced Edition

A full-stack student management system with secure custom authentication, role-based access control (RBAC), and comprehensive API endpoints for user and student data management.

## 🚀 Features

- **Custom Credentials Authentication**: Email/password login with bcrypt password hashing
- **Role-Based Access Control (RBAC)**: Two-tier access system (USER, ADMIN)
- **Secure API Endpoints**: Protected routes with role-based authorization
- **User Management**: Admin dashboard for managing users and roles
- **Student Management**: Track students with full CRUD operations
- **Session Management**: JWT-based stateless sessions
- **TypeScript Support**: Fully typed codebase for better developer experience
- **Vercel Ready**: Configured for easy deployment

## 🛠️ Technology Stack

- **Frontend**: Next.js 16.1.6, React, TypeScript, Tailwind CSS
- **Backend**: Next.js API Routes, Next-auth v5
- **Database**: SQLite with Prisma ORM
- **Authentication**: NextAuth.js with custom CredentialsProvider
- **Security**: bcryptjs for password hashing
- **Styling**: Tailwind CSS

## 📋 Prerequisites

- Node.js 18+ and npm
- SQLite (bundled with better-sqlite3 adapter)
